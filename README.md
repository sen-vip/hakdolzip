# 📋 행정실 루틴집

**(매월 반복 업무 요약)**

학교 행정실에서 매월 반복적으로 확인해야 하는 업무를 한눈에 정리한 웹페이지입니다.  
초보자도 “이번 달에 뭘 확인해야 하지?”를 빠르게 파악할 수 있도록 만든 **행정실 반복업무 요약 자료**입니다.

---

## 🔗 바로가기

- 웹페이지: https://sen-vip.github.io/hakdol-routine
- GitHub: https://github.com/sen-vip/hakdol-routine

---

## ✨ 주요 기능

### ✅ 확인 체크 기준월

업무별로 확인해야 하는 기준월을 체크 표시로 보여줍니다.  
월별로 챙겨야 할 업무를 놓치지 않도록 시각적으로 확인할 수 있습니다.

### 🔍 검색 기능

업무명, 설명, 메모에 포함된 단어를 검색해 필요한 업무를 빠르게 찾을 수 있습니다.

### 🏷️ 분류 필터

업무 성격별로 분류를 선택해 필요한 업무만 골라볼 수 있습니다.  
반복업무가 많아도 원하는 항목을 쉽게 좁혀볼 수 있습니다.

### 🗂️ 업무 카드 정리

각 업무는 카드 형태로 정리되어 있습니다.

- 업무 제목
- 일정
- 분류
- 설명
- 메모
- 확인 체크 기준월

### ⬆️ 상단 고정바

페이지 상단에 제목, 버전, GitHub 링크, 맨위로 이동 버튼을 배치했습니다.  
아래로 내려가도 필요한 이동 기능을 쉽게 사용할 수 있습니다.

## 👩‍💼 이런 분께 좋아요

- 행정실 업무를 처음 맡은 신규 담당자
- 매월 반복업무를 놓치지 않고 확인하고 싶은 담당자
- 업무 인수인계 자료가 필요한 행정실
- 월별·분기별·연간 업무 흐름을 빠르게 파악하고 싶은 사람
- “이 업무는 언제 챙겨야 하지?”를 자주 확인하는 사람

## 🎨 디자인 방향

전체 디자인은 행정실에서 부담 없이 사용할 수 있도록 밝고 가벼운 업무도구 느낌으로 구성했습니다.

- 밝은 배경
- 부드러운 테두리
- 과하지 않은 그림자
- 읽기 쉬운 글씨 크기
- 업무 제목과 일정 강조
- 초보자도 보기 쉬운 카드형 구성
- 체크 기준월이 눈에 잘 들어오는 구조

---

## 🧭 사용 방법

1. 웹페이지에 접속합니다.
2. 검색창에 찾고 싶은 업무명을 입력합니다.
3. 필요한 경우 분류 필터를 선택합니다.
4. 업무 카드에서 일정, 설명, 메모를 확인합니다.
5. 확인 체크 기준월을 보고 해당 월에 챙겨야 할 업무를 점검합니다.

## 📝 제작 의도

**행정실 루틴집**은 반복업무를 `업무 제목 + 일정 + 설명 + 메모 + 확인 기준월` 중심으로 정리해, 업무 흐름을 빠르게 익히고 놓치는 일을 줄이기 위해 제작했습니다.

특히 신규 담당자나 업무 인수인계를 받는 사람이 별도의 긴 설명 없이도 “언제 확인해야 하는지”, “무엇을 챙겨야 하는지”를 바로 볼 수 있도록 구성했습니다.

다만 이 자료는 실제 업무 처리를 대신하는 기준 문서가 아니며, 최종 업무 처리는 반드시 관련 법령·지침·공문·기관 기준을 확인해 진행해야 합니다.
