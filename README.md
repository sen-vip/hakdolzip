# 🗂️ 학교 행정실 반복업무 요약zip

> 매월·분기·연간 단위로 반복되는 학교 행정실 업무를 한눈에 정리하는 웹앱입니다.

학교 행정실에서 반복적으로 확인해야 하는 업무를
월별, 분기별, 연간, 수시 업무로 나누어 빠르게 살펴볼 수 있도록 만든
**행정실 반복업무 요약 도구**입니다.

업무 일정, 분류, 상세내용, 메모를 한 화면에서 확인하고
필요할 때 CSV로 내보낼 수 있습니다.

## 🔗 바로가기

https://sen-vip.github.io/hakdol_routine.zip/

## ✨ 주요 기능

### 1. 반복 업무 한눈에 보기

매월 반복 업무, 분기/연간 업무, 수시 업무를 구분하여
학교 행정실에서 챙겨야 할 업무 흐름을 빠르게 확인할 수 있습니다.

### 2. 카테고리별 확인

업무 성격에 따라 카테고리별로 업무를 모아볼 수 있습니다.
필요한 업무만 골라서 확인할 때 유용합니다.

### 3. 주기별 확인

업무 주기에 따라 내용을 확인할 수 있습니다.

* 매월 반복 업무
* 분기 업무
* 연간 업무
* 수시 업무

### 4. 정렬 기능

업무 목록을 상황에 맞게 정렬할 수 있습니다.

* 기본 순서
* 일정 빠른 순
* 카테고리 순
* 업무명 가나다순

### 5. CSV 내보내기

화면에 정리된 반복 업무 목록을 CSV 파일로 내보낼 수 있습니다.
엑셀에서 열어 추가 정리하거나 공유용 자료로 활용할 수 있습니다.

## 💡 이런 상황에서 사용하면 좋아요

* 매월 반복되는 행정실 업무를 놓치지 않고 확인하고 싶을 때
* 분기별, 연간 업무를 미리 정리하고 싶을 때
* 신규 직원이나 업무 인수인계용으로 반복 업무 흐름을 설명해야 할 때
* 행정실 업무 체크리스트의 기초 자료가 필요할 때
* 엑셀로 업무 목록을 내려받아 따로 관리하고 싶을 때

## 🧭 화면 구성

| 영역       | 설명                                        |
| -------- | ----------------------------------------- |
| 상단 요약    | 전체 업무 수, 매월 반복 업무, 분기/연간 업무, 수시 업무를 요약 표시 |
| 카테고리     | 업무 분야별로 필터링                               |
| 주기       | 업무 반복 주기별로 필터링                            |
| 정렬       | 일정, 카테고리, 업무명 기준으로 목록 정렬                  |
| 업무 목록    | 일정, 분류, 업무 및 상세내용, 메모 확인                  |
| CSV 내보내기 | 현재 업무 목록을 CSV 파일로 저장                      |

## 📝 사용 방법

1. 웹페이지에 접속합니다.
2. 필요한 경우 카테고리나 주기를 선택합니다.
3. 정렬 기준을 선택해 업무 목록을 보기 좋게 정리합니다.
4. 필요한 업무의 상세내용과 메모를 확인합니다.
5. 별도 정리가 필요하면 `CSV 내보내기`를 눌러 파일로 저장합니다.

## 🔒 저장 및 보안 안내

이 웹앱은 GitHub Pages에서 실행되는 정적 웹페이지입니다.
별도의 서버 저장 기능이나 로그인 기능 없이 사용할 수 있습니다.

다만 CSV로 내보낸 파일은 사용자의 PC에 저장되므로,
학교 내부 업무 자료를 포함하는 경우 파일 관리에 유의해 주세요.
